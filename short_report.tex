\begin{abstract}
This report evaluates a DST-based classification pipeline that learns evidential masses on top of rule sets induced by FOIL and RIPPER. The benchmark uses one fixed train/test split per dataset, five training seeds, and nine tabular datasets. In this setting, learned evidential aggregation is often competitive with simpler rule-based aggregation on the same induced rule sets. The result should be read as a benchmark summary for this evaluated setup, not as a broad generalization claim.
\end{abstract}

\section{Benchmark Setup}
The current experimental setup has two stages. First, FOIL or RIPPER induces a candidate rule pool, and optional post-growth shaping filters it for quality, redundancy, and class balance. Second, DSGD learns evidential masses on top of the resulting induced rule set.

Earlier project runs also explored a separate internal STATIC rule source, where rules were generated automatically from single literals and short conjunctions. The present report focuses on the induced-rule setting, where the rule sets come from FOIL or RIPPER rather than from that earlier automatic construction scheme.

The benchmark compares several prediction rules on the same induced rule sets. \texttt{native\_ordered\_rule} predicts from the first fired rule in the learned order. \texttt{first\_hit\_laplace} uses that first rule together with its Laplace-smoothed class estimate. \texttt{weighted\_vote} aggregates all fired rules heuristically. \texttt{dsgd\_dempster} learns evidential masses and combines the fired rules with Dempster's rule.

The comparison therefore isolates different ways of aggregating predictions from the same induced rule sets.

\section{Hard Cases}
To examine where learned aggregation may help, we also study a hard-case subset of the test split. This subset contains samples where the fired rules show conflict, weak support for the true class, or an unusually ambiguous voting pattern for the same dataset/inducer pair.

This analysis is diagnostic rather than causal. In the current benchmark snapshot, learned masses help on some hard-case subsets and do not help on others. The appropriate takeaway is therefore narrow: evidential aggregation can be useful on some conflict-heavy samples, but the effect is not uniform across all dataset/inducer pairs.

\section{Conclusion}
On the present fixed-split five-seed benchmark, learned evidential aggregation on FOIL- and RIPPER-induced rules is often competitive with simpler rule aggregation on the same induced rule sets. The evidence supports a narrow benchmark conclusion for this evaluated setup, but it does not support a broader claim of general superiority or a mechanistic explanation of performance differences.
